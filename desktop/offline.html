No Slip Quant 대시보드 연결 대기

localhost:3000 응답을 기다리는 중 — 서버가 켜지면 자동으로 연결됩니다 (5초 간격 재시도)

1
저장소 폴더에서 대시보드 서버 시작 cd noslip_quant_bot && npm run dev
2
(선택) 텔레그램 봇/모니터 데몬 시작 services/trader/.venv/bin/python services/trader/telegram_interactive_bot.py
3
원격 서버를 쓰는 경우 환경변수로 실행 NOSLIP_DASHBOARD_URL=https://your-host open -a "No Slip Quant"
메뉴 Dashboard → "새로고침 / 재연결" (⌘R)로 즉시 재시도할 수 있습니다.
